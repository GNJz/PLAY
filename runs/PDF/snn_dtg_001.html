Qquartz + SNN + DTG 기반 Cookiie AI Brain (Draft)

원페이지 개요: Qquartz 4-state 연산 모델과 SNN + DTG를 결합한 초저전력/감성-인지 통합 브레인.
데모 타깃: 간단 SNN 실험(IG on/off) → 우주선 시뮬레이션(GNC/3-body/Navier–Stokes)로 확장 가능한 전체 아키텍처 제안.
상태: Draft (공개발표용 초안) • 재현 태그: IDEA / DRAFT / VERIFIED 중 DRAFT

1. Abstract

본 초안은 Qquartz(4상태: -1/0/+1/+2) 연산 모델을 SNN(Spiking Neural Network) 및 DTG(Dynamic Threshold Gate)와 결합해 초저전력·감성/인지 통합 브레인(Cookiie Brain)을 제안한다.
미니 데모로 5뉴런 LIF 네트워크에서 임계값 게이팅(IG) 효과(발화 전이)를 보여주고, 이를 우주선 시뮬레이션(3-body, Navier–Stokes, GNC 루프)까지 확장하는 전체 아키텍처의 밑그림을 제시한다.

2. 배경과 문제정의

  • 기존 AI는 이진(0/1) 중심 → 감성/맥락 신호 표현력·에너지 효율의 한계.
  • 우주선/로보틱스는 초저전력 실시간 의사결정이 필요.
  • SNN은 이벤트 기반·저전력에 강점. 여기에 DTG로 임계값을 상황(맥락/감정/에너지)별로 동적으로 조절하면 반응성이 크게 바뀐다.
  • Qquartz는 4상태 전위를 통해 감성/맥락 신호를 값 영역에 직접 매핑하려는 시도.

3. 핵심 아이디어

  • Qquartz: -1, 0, +1, +2 네 상태를 이온 전위(Ca/Na/K/Cl) 영감으로 정의해 신호 표현력↑, 전력↓.
  • SNN + DTG: 발화 임계값을 동적으로 조정해 무발화 → 집단발화 전이를 유도, 감응성 튜닝.
  • Cookiie Brain: 위 결합을 하나의 감성/인지 통합 스택으로 묶어, 로봇/우주선 시뮬 의사결정에 적용.

4. 방법(Method)

4.1 Qquartz Computing

  • 상태: -1/0/+1/+2 (이온 전위 영감)
  • 목적: 감성/맥락 신호를 정수 상태로 저손실 인코딩 → 이벤트 기반 SNN과 자연 결합

4.2 SNN (LIF 예시)

τ dV/dt = -(V - V_rest) + R·I(t)
if V ≥ θ → spike; V ← V_reset

학습: STDP/강화학습 결합 실험 예정

4.3 DTG (Dynamic Threshold Gate)

θ_{t+1} = (1-λ)θ_t + λ(b + α·E_t - β·I_t)
  • E_t: 맥락/에너지/감성 신호
  • I_t: 억제 신호
  • α, β, λ, b: 튜닝 파라미터

4.4 우주선 시뮬레이션 통합

  • 3-body: 복잡 궤도/섭동 반응 실험
  • Navier–Stokes: 추력/유동 모델(엔진/자세제어)
  • GNC: Guidance–Navigation–Control 루프에 정책 연결

5. 데모 결과(요약)

실험: 5뉴런 LIF, 1s, dt=1ms, 외부 펄스(0.1–0.7s)
비교: Baseline(α=1.0) vs IG on(α=0.7; 임계값 30% 완화)
